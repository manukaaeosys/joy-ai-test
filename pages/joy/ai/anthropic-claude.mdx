## Anthropic Claude

[Claude](https://claude.ai/), developed by [Anthropic](https://www.anthropic.com/), is a next-generation AI assistant designed to be helpful, honest, and harmless. It's accessible through a chat interface and API, capable of a wide variety of conversational and text processing tasks. 

Claude is particularly noted for its high degree of reliability and predictability. It supports various use cases including summarization, search, creative writing, Q&A, coding, and more. Claude is available in two versions: the high-performance Claude and the faster, lighter Claude Instant. ## How to use Claude models in Joy AI

### Get an API Key

- Visit the Claude Website: Go to the [Claude website](https://claude.ai/) provided by Anthropic.
- Request Access: Look for an option to request access to Claude. This may involve filling out a form or expressing your interest in using Claude's API.
- Wait for Approval: After submitting your request, wait for approval or further instructions from Anthropic.
- API Key Generation: Once approved, you'll likely receive instructions on how to generate your API key from the Claude platform.

### Use Your API Key

You can now use this API key in your applications to access the various LLM models offered by OpenRouter.

- In Joy, go to **Settings** -> **Advanced** -> **AI**
- Select **Anthropic Claude**
- Enter your API Key and a model of your choice
