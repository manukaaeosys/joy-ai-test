## Google AI Gemini 

[Google AI Gemini](https://deepmind.google/technologies/gemini) is a technology developed by Google's DeepMind, focusing on advanced artificial intelligence research and applications. It's part of Google's suite of AI technologies designed to push the boundaries of machine learning and AI capabilities. Gemini emphasizes building intelligent systems that can understand, learn, and adapt, contributing to a wide range of practical and innovative applications. 

![](https://user-images.githubusercontent.com/2284279/291581949-192fefa2-7d18-47b9-bac1-7510ae9a1bb7.png)

## How to use Gemini models in Joy AI

### Create an Account

- Go to [Google AI Studio](https://makersuite.google.com)

### Create an API Key

- Navigate to **Get API Key** section
- Choose to create new API Key for new or existing project

![](https://user-images.githubusercontent.com/2284279/291582583-a35a752f-be00-46f0-82c0-13e398e90fd6.png)

### Use Your API Key

You can now use this API key in your applications to access the various LLM models offered by OpenRouter.

- In Joy, go to **Settings** -> **Advanced** -> **AI**
- Select **Google Gemini**
- Enter your API Key and a model of your choice
