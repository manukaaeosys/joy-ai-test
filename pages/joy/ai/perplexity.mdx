## Perplexity

[Perplexity AI](https://www.perplexity.ai/) is an innovative platform designed to revolutionize the way we interact with information. 

It combines advanced natural language processing with a user-friendly interface, enabling users to ask complex questions and receive detailed, accurate answers. This platform is particularly notable for its ability to understand and process a wide range of queries, offering insights and explanations that go beyond basic search results. Perplexity AI aims to make information access more intuitive and effective, catering to a diverse range of information needs and enhancing the overall experience of information discovery.

## How to use Perplexity models in Joy AI

### Create an Account

- Create an account at [Perplexity](https://www.perplexity.ai/)

### Create an API Key

- Log In: Once you're logged in, go to [Account Settings](https://www.perplexity.ai/settings/api)
- Navigate to API Keys section and hit **Generate**it **Generate a new key** to create new API Key

![](https://user-images.githubusercontent.com/2284279/291579524-b8e16d25-305e-4908-9eb1-d2f0c726f2c9.png)

### Use Your API Key

You can now use this API key in your applications to access the various LLM models offered by OpenRouter.

- In Joy, go to **Settings** -> **Advanced** -> **AI**
- Select **Perplexity**
- Enter your API Key and a model of your choice
