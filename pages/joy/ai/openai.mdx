## OpenAI

OpenAI is an artificial intelligence research lab consisting of the for-profit corporation OpenAI LP and its parent company, the non-profit OpenAI Inc. The organization is notable for its research in AI and its creation of advanced AI models. OpenAI has developed various AI technologies, including the well-known language processing model GPT (Generative Pre-trained Transformer) and the image generation model DALL-E

OpenAI Chat is a conversational interface that allows users to interact with OpenAI's advanced language models. It's designed to understand and respond to a wide range of queries, enabling users to engage in natural, human-like conversations

## How to use Open AI models in Joy AI

### Create an Account

- Create an account at [Open AI](https://www.openai.com/)

### Create an API Key

- Log In: Once you're logged in, go to [Platform](https://platform.openai.com/api-keys)
- Navigate to API Keys section and hit **Create new Secret Key** to create new API Key

![](https://user-images.githubusercontent.com/2284279/291580876-622ebc8d-95a5-495b-9af9-6804c8ab3de2.png)

### Use Your API Key

You can now use this API key in your applications to access the various LLM models offered by OpenRouter.

- In Joy, go to **Settings** -> **Advanced** -> **AI**
- Select **Open AI**
- Enter your API Key and a model of your choice

![](https://user-images.githubusercontent.com/2284279/291584162-e044ebb7-ad77-4854-9064-59bf21f5fec6.png)
