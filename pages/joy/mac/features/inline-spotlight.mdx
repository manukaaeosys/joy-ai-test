## Inline Spotlight

Joy's Inline Spotlight mode is a hybrid feature that combines elements of both Spotlight and Inline modes. With this feature, users can activate the Spotlight interface to start a conversation and then seamlessly paste the generated content inline into the current application they are using. 

This mode offers a streamlined and efficient way to incorporate AI-generated responses directly into users' workflows, enhancing productivity and convenience.

## How to

- Go to Joy **Settings** -> **Keyboard Shortcuts**
- Assign a keyboard shortcut for **Inline Spotlight**
